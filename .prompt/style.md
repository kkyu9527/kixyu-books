# Style Rules

本主题的整体方向是：极简、安静、克制、书籍化、强调连续阅读。

请始终遵守以下规则：

1. 所有设计都优先服务于文字内容和阅读体验。
2. 整个主题应更像一本持续更新的数字书，而不是传统博客门户。
3. 弱化分类、标签、归档的展示感，强化目录、章节、前后文关系。
4. 避免信息流、卡片流、宫格式堆砌，不要追求页面热闹。
5. 避免高饱和色彩、复杂装饰、强视觉刺激和无关动画。
6. 导航、提示、交互都应保持轻量，不打断阅读。
7. 文案与命名应偏阅读语境，避免营销化、运营化表达。
8. 所有新增实现都应先判断：是否更有利于“像一本书一样阅读”。
9. 如果一个设计增强了展示感但削弱了阅读感，应放弃。
10. 未经明确要求，不要加入与整体气质不一致的模块和表达。
